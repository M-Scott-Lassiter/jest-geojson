(static) feature(featureObject, geometryTypeopt) → {boolean}

Source:

See:
  • https://github.com/M-Scott-Lassiter/jest-geojson/issues/24
Verifies an object is a valid GeoJSON Feature. This object requires a "type" member that must equal 'Feature', a "geometry" member that contains either one of the seven valid GeoJSON geometry objects or an empty array, and a "properties" member that is either an object of any composition or null. Foreign members are allowed with the exceptions thrown below. If present, bounding boxes must be valid.
Example
const testFeature = {
    "type": "Feature",
    "bbox": [-10.0, -10.0, 10.0, 10.0],
    "geometry": {
        "type": "Polygon",
        "coordinates": [
            [
                [-10.0, -10.0],
                [10.0, -10.0],
                [10.0, 10.0],
                [-10.0, -10.0]
            ]
        ]
    },
    "properties": {
        "prop0": "value0",
        "prop1": {
            "this": "that"
        }
    }
}
const multiPoint = {
    type: "MultiPoint",
    coordinates: [
        [101.0, 0.0],
        [102.0, 1.0]
    ]
}

const goodExample1 = feature(testFeature)) // true
const goodExample2 = feature(testFeature, 'Polygon')) // true

const badExample1 = feature(multiPoint)) // throws error
const badExample2 = feature(testFeature, 'LineString')) // throws error
const badExample3 = feature(testFeature.geometry, 'Polygon')) // throws error
